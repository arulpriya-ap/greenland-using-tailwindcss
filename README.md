# 🌿 Greenland Website

A modern and responsive plant shopping website developed using **HTML** and **Tailwind CSS**. The project consists of three fully designed pages: **Home**, **Products**, and **Contact**. The website is built with a clean user interface and responsive design to provide a smooth user experience across all devices.

---

# 📌 Project Overview

The Greenland website is designed for showcasing and selling plants online. The website focuses on responsive layouts, modern UI design, and user-friendly navigation. The project demonstrates the practical usage of Tailwind CSS utilities, responsive design concepts, Flexbox, and Grid layouts.
